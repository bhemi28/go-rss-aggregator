# Go RSS Aggregator

The Go RSS Aggregator is a server application designed for managing RSS feeds and user subscriptions. Built using Go, this project enables users to create feeds, subscribe to them, and retrieve personalized feed data.

---

## Features

- **Create Users:** Easily create user profiles with API keys for authentication.
- **RSS Feed Management:** Users can create, subscribe to, and manage RSS feeds.
- **Authentication Middleware:** Includes middleware for handling API key-based authentication.
- **Database Integration:** Persistent storage using PostgreSQL via `sqlc`.
- **Custom Endpoints:** API endpoints for readiness checks, user creation, feed subscription management, and more.

---

## Installation

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/bhemi28/go-rss-aggregator.git
   cd go-rss-aggregator
   ```

2. **Set Up Your Environment:**
   - Create a `.env` file in the root directory with the following configuration:
     ```
     PORT=8080
     DB_URL=your_postgresql_connection_string
     ```

3. **Install Dependencies:**
   Make sure you have Go installed, then run:
   ```bash
   go mod tidy
   ```

4. **Run the Application:**
   ```bash
   go run main.go
   ```

## API Endpoints

### User Management
- **Create User:**
  - URL: `POST /v1/user`
  - Payload:
    ```json
    {
      "name": "example-user"
    }
    ```

- **Get User:**
  - URL: `GET /v1/user`
  - Header: `Authorization: Bearer <API_KEY>`

### RSS Feeds
- **Create Feed:**
  - URL: `POST /v1/feed`
  - Payload:
    ```json
    {
      "title": "Feed Title",
      "url": "https://example.com/rss"
    }
    ```

- **Subscribe to Feed:**
  - URL: `POST /v1/feed/add?feed_id=<FEED_ID>`
  - Header: `Authorization: Bearer <API_KEY>`

- **Get Feeds:**
  - URL: `GET /v1/user_feeds`
  - Header: `Authorization: Bearer <API_KEY>`

## Project Structure

- **`main.go`:** Initializes the application, sets up routes, and starts the server.
- **`handler_user.go`:** User-related handlers (create, get user).
- **`handler_feed.go`:** Feed-related handlers (create, subscribe to feeds).
- **`model.go`:** Defines types and conversion functions for users, feeds, and subscriptions.
- **`middleware.go`:** API key-based authentication middleware.
- **`internal/`:** Contains database queries generated by `sqlc`.

- **Language:** Go

  - `lib/pq`: PostgreSQL driver.
  - `godotenv`: Environment variable management.
- **Database:** PostgreSQL
